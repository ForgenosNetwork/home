About the Simulator

This tool is an interactive Git conflict simulator designed for developers to practice resolving complex merge conflicts in a safe, visual environment. It simulates real-world branch divergence and provides a robust testing ground for conflict resolution strategies.

Key Features

  • Visual Branch Graph: Isometric 3D rendering of Git commit history and branch paths.
  • Real-time Code Diffs: Side-by-side terminal viewer showing HEAD and incoming changes.
  • Multiple Resolution Strategies: Practice accepting current, incoming, or manual combinations.
  • 100% Secure & Local: All processing happens in your browser—no code is sent to servers.

How to Use

  1. 1
    Click "Initiate Merge Simulation" to generate a complex Git divergence scenario.
  2. 2
    Trigger the simulated merge and observe the conflict node pulsing on the branch graph.
  3. 3
    Review the conflict blocks (HEAD vs. feature) in the terminal diff viewer.
  4. 4
    Select your preferred resolution strategy using the control panel to complete the merge.

Who It's For

Junior developers mastering Git workflows, students in coding bootcamps, and team leads training their members on proper conflict resolution and branch weaving.
